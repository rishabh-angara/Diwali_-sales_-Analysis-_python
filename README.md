# Diwali_-sales_-Analysis-_python 
📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on Diwali sales data using Python.
The objective is to analyze customer purchasing behavior and identify key insights that can help businesses improve customer experience and increase sales during festive seasons.

The analysis focuses on:

Customer demographics
Gender-based purchasing trends
Age group analysis
State-wise sales performance
Product category insights
Marital status and occupation analysis

📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on Diwali sales data using Python.
The objective is to analyze customer purchasing behavior and identify key insights that can help businesses improve customer experience and increase sales during festive seasons.

The analysis focuses on:

Python

Gender vs Purchasing Power

Product Category Trends
Marital Status Analysis
3. Visualization

Created various charts including:

Bar plots
Count plots
Heatmaps
Distribution plots
📈 Major Insights
Female customers contributed more to overall sales.
Customers aged 26–35 years purchased the most.
Married women from states like Uttar Pradesh, Maharashtra, and Karnataka were major buyers.
IT, Healthcare, and Aviation professionals spent the highest amounts.
Food, Clothing, and Electronics categories had strong sales performance.
